Cores

Podemos destacar cores em nosso site usando escala hexadecimal.

O azul pode ser representado pela tag BACKGROUND-COLOR, pela escala (#0000ff), pela representação RGB(0, 0, 255), ambos gerando os mesmos valores

Criando degradê

Para criar um degrade, usamos a tag BACKGROUND-IMAGE: LINEAR-GRADIENT, acompanhada do comando de direnção e cores em parentes.

Identificações

Em HTML se identifica como id, em CSS #. (so pode um elemento)

Em HTML é class, em CSS é ponto '.'.(pode varios elementos)

TAG HOVER

Usada para manipular seletores na tag DIV. (div=hover)
