Negrito e Itálico

Para usar negrito, usamos o comando STRONG.

E para usar italico, usamos o comando EM.

Marcações:

O comando usado para marcação é MARK.

Podemos acrescentar um estilo diferente de marcação usando o comando STYLE, seguido de "background-color".

Exemplo: 'Fui ao mercado hoje.' (selecionei ao mercado com o comando descrito acima.)

Texto excluido:

Para texto excluido/deletado, usamos a tag DEL.

Texto inserido ou sublinhado:

Para sublinhar um texto, usamos a tag INS.

Texto sobrescrito:

Para sobrescrever algum valor, como elevar um numero ao quadrado ou cubo... usamos a tag SUP.

Exemplo: x20+3 (sem tag) / x20+3 (com tag).

Texto subscrito:

Para sobscrever algum valor, usamos a tag SUB.

Exemplo: H2O (sem tag) / H20 (com tag).

Outros comandos:

Melhorar qualidade da escrita para facilitar leitura: tag CODE.

Frase usando a tag acima. Note a diferença.

Envelopar um conjunto de escrita: tag PRE.

Citações:

Para usar aspas ("), usamos a tag Q.

Como diria um velho amigo, o computador é um burro muito veloz.

Abreviações:

Para abreviar algo, usamos a tag ABBR.

Exemplo: Estou adorando este curso do CeV.

A tag ABBR foi utilizada em "CeV".

Texto invertido:

Para texto invertido, usamos a tag BDO.

Usaremos o dir:"rtl".

Exemplo: Estou adorando este curso.
